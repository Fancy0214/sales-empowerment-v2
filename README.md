# 留学销售赋能平台 V2

## 项目简介

基于 Fancy 现有 B2B 留学销售赋能平台打造的全新升级版本，提供完整的销售支持工具集。

## 功能模块

| 模块 | 功能描述 |
|------|---------|
| 📊 仪表盘 | 欢迎页、关键指标、漏斗图、业绩趋势、快速入口 |
| 🤖 AI话术工坊 | 智能生成定制化销售话术，支持多场景 |
| 🔍 竞品情报站 | 竞品对比、雷达图分析、动态追踪 |
| 🎓 院校数据库 | 12所全球顶尖院校信息，智能筛选 |
| 🔁 客户漏斗 | 6阶段漏斗可视化，实时跟进管理 |
| 💬 话术库 | 17+精选话术，分类管理，一键复制 |
| 🧰 销售工具箱 | 5A模型、客户画像、报价模板等 |
| 🎯 九型人格 | 完整9型详解、互动测试、团队搭配 |
| 📚 实战案例库 | 成功案例与失败复盘，深度解析 |

## 技术栈

- 纯静态 HTML/CSS/JavaScript
- 单页应用 (SPA) 架构
- Font Awesome 图标库
- 响应式设计，移动端友好

## 快速开始

```bash
# 本地预览
open index.html

# 或使用 Python 服务器
python -m http.server 8080
# 访问 http://localhost:8080
```

## 部署

本项目配置为 GitHub Pages 部署，push 到 main 分支后自动生效。
